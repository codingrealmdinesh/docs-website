## Set up the Github integration [#set-up-github]

Complete the following steps to set up the Github integration:

<Steps>
<Step>
## Select the correct organization [#org-select]

To make the github integration work, you need to select the desired organization of your choice    
</Step>
<Step>

## Integration Details [#integration-details]

By default, the Power Service catalog is selected, which will ensure to import all the data from Teams, Repositories and Users.
    </Step>

    <Step>

## Github Integration [#github-integration]

1. In this page, once we click on the install button, we get redirected to the Github App installation page. From there we need to install the Github app into the organization.
2. If we are successfully able to install, it will show a success message with the teams, repo and users lists or else a failure message.
    </Step>
    
</Steps>
